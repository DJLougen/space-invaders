# Space Invaders

A classic Space Invaders game built with vanilla HTML5 Canvas and JavaScript. No dependencies, no build step — just open the file and play.

**[Play now](https://djlougen.github.io/space-invaders/)**

## Controls

| Key | Action |
|-----|--------|
| `← →` or `A D` | Move ship |
| `Space` or `↑` | Shoot |
| `P` | Pause |
| `Enter` | Restart (after game over) |

## Features

- 5 rows of invaders across 3 types (10/20/30 pts)
- Invaders speed up as their numbers thin out
- Mystery UFO with random bonus (50–300 pts)
- 4 destructible shields that degrade from both sides' fire
- Particle effects on kills
- Progressive difficulty across levels
- Shields carry over between levels

## Running locally

Open `index.html` in any browser. No server needed.
